DJANGO 时间少8小时

DJANGO 时间少8小时

今天打开博客发现时间差8小时，马上查看服务器时间发现时间是正确的。最后查看setting配置文件原来时区默认设置UTC，更改即可正常显示

vim blogs/settings.py
把TIME_ZONE = ‘UTC’更为为以下
TIME_ZONE = ‘Asia/Shanghai’
