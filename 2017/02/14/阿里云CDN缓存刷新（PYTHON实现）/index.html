阿里云CDN缓存刷新（PYTHON实现）

阿里云CDN缓存刷新（PYTHON实现）
项目背景：
由来：由于作者公司把Ucloud云服务器迁移到阿里云上，并使用了阿里的OSS文件系统，出于成本考虑并使用上阿里的CDN服务。但是迁移后，开发隔三两日就跑过来叫帮忙刷新缓存。当时思想，每次这样搞并发长久之策，便尝试编写缓存系统。
编写过程
1、查阅官方API文档
https://help.aliyun.com/document_detail/27200.html?spm=5176.doc27201.6.624.B0NT84
2、选择前端框架
觉得最快实现还是选择Bootstrap
3、选择后端框架
选择基于PYTHON的DJANGO框架
